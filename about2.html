เซต (ที่ต้องรู้)

เซต (set) คือ

        เหตุการณ์ที่สนใจศึกษา เช่น A = {1,2,3} –> 1,2,3 เป็นสมาชิกในเซต A

รูปแบบเซต

        1. แจกแจงสมาชิก เช่น A = {1,2,3}
        2. แบบบอกเงื่อนไข เช่น { Xl X>3 และ X < 10 }

สิ่งที่ควรรู้

         เซตว่าง = ∅ ( เซตที่ไม่มีสมาชิก )
         เซตจำกัด – > {1,2,7}
         เซตอนันต์ – > {1,2,…} มีสมาชิกแบบไม่สิ้นสุด
         สับเซต คือ เซตย่อยของสิ่งนั้น ๆ แทนด้วย ⊂
         n(A) = จำนวนสมาชิกในเซต A (การนับจำนวนสมาชิกจะไม่นับตัวซ้ำ)
เช่น A = {1,5,8,8,8,9,{9}}
n(A) = 5
         พาวเวอร์เซต P(A) = เซตของสับเซตทั้งหมด
*เซตว่าง ∅ เป็นสับเซตของทุกเซต
เช่น B = {1,7}
P(B) = {{1},{7},{1,7},∅}
         เอกภพสัมพัทธ์ (U) = เซตที่กำหนดเพื่อกล่าวถึงสิ่งที่เป็นสมาชิกของเซตนี้เท่านั้น
เช่น U = {1,2,3}
A = {Xl X > 2}
